## Getting Started

### Backend

Python Flask server with WebSocket support.

```bash
cd server
pip install -r requirements.txt
python application.py
```

### Frontend

React/TypeScript with webpack.

```bash
npm install
npm start
```

### Tests

```bash
npm test
```
